创建一个 Chrome Extension 项目，这个项目为了解决一个痛点：在浏览 x.com 的 posts 的时候，方便地知道这个用户是谁。这个插件需要在

@example.png 的用户展示区域添加一个描述这个用户是谁的文案信息。数据源来自 https://github.com/maoxiaoke/who-are-you-x/blob/main/data.json。其结构为：

```json
[
  {
    "account": "@ciguleva",
    "who": "Perplexity.ai 的创意总监"
  }
]
```

逻辑为匹配到 `account` 的用户，在用户的信息的右侧展示 `who`。
